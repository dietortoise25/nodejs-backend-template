- 项目初始化

  1. pnpm init

  2. 修改 package.json 中的 type 为 module

  3. 创建 `src/server.ts`和`src/app.ts`

  4. TypeScript 安装

     ```bash
     pnpm add typescript tsx
     ```

  5. TypeScript 配置

     1. 安装 TypeScript 相关依赖
        ```bash
        pnpm add -D @tsconfig/node-lts
        ```
     2. 执行 `pnpm tsc --init` 创建 tsconfig.json 文件

     ```json
     {
       "extends": "@tsconfig/node-lts/tsconfig.json",
       "compilerOptions": {
         "allowImportingTsExtensions": true, // 允许导入 .ts 文件
         "noEmit": true // 不生成 js 文件
       }
     }
     ```

  6. dev 命令配置
     - 在 package.json 中添加 dev 脚本 (tsx watch 模式已经覆盖热加载功能)
     - 创建./src/server.ts 文件,随便写点东西打印
     - 安装 dotenvx `pnpm add @dotenvx/dotenvx`
     - 创建`.env`文件
     - 配置`package.json`中的 dev 命令
       ```json
       {
         "scripts": {
           "dev": "dotenvx run -- tsc --noEmit && dotenvx run -- tsx watch ./src/server.ts"
         }
       }
       ```
